HOW TO PLAY

* Use **Arrow Keys (↑ ↓ ← →)** to move tiles
* When two tiles with the same number collide, they merge into one
* After each move, a new tile (2 or 4) appears
* Reach **2048** to win (or continue for higher scores!)
